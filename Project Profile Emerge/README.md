# Pradhan's Professional Portfolio

A modern, futuristic, and fully responsive full-stack engineer portfolio website showcasing skills, projects, and professional experience.

## 🌟 Features

### Design & UX
- **Modern Design**: Inspired by contemporary portfolio sites like ioi.dk
- **Futuristic Aesthetic**: Gradient backgrounds, glassmorphism effects, smooth animations
- **Fully Responsive**: Optimized for mobile, tablet, and desktop devices
- **Dark Theme**: Professional dark color scheme with vibrant cyan and purple accents
- **Smooth Animations**: Scroll animations, hover effects, and interactive elements

### Sections
- **Hero Section**: Eye-catching introduction with floating cards and interactive elements
- **About**: Professional background with statistics and skill sphere visualization
- **Skills**: Organized by category (Frontend, Backend, Database, DevOps, Tools)
- **Projects**: Six featured projects with detailed descriptions and tech stacks
- **Experience**: Professional timeline showing career progression
- **Contact**: Forms, contact information, and social media links

### Technology Stack
- **Frontend**: HTML5, CSS3 (with advanced features like CSS Grid, Flexbox, Custom Properties)
- **JavaScript**: Vanilla JS with modern ES6+ features
- **Libraries**: 
  - AOS (Animate On Scroll) - For scroll animations
  - Font Awesome - For icons
  - Google Fonts - For typography
- **No Frameworks**: Pure HTML, CSS, and JavaScript for maximum performance

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No build tools or dependencies required

### Installation

1. Clone or download the project:
```bash
git clone https://github.com/pradhan/portfolio.git
cd portfolio
```

2. Open `index.html` in your browser or use a local server:
```bash
# Using Python 3
python -m http.server 8000

# Using Python 2
python -m SimpleHTTPServer 8000

# Using Node.js (with http-server)
npx http-server
```

3. Navigate to `http://localhost:8000` in your browser

## 🎨 Customization

### Update Personal Information

1. **In `index.html`**:
   - Update hero section title and subtitle
   - Change email, phone, and location in contact section
   - Update social media links
   - Modify content in about, skills, projects, and experience sections

2. **In `style.css`**:
   - Customize color variables in `:root` for brand colors
   - Adjust animations timing and effects
   - Modify spacing and typography

3. **In `script.js`**:
   - Update form submission handler
   - Modify notification messages
   - Add analytics integration

### Key CSS Variables

```css
:root {
    --primary: #00d4ff;        /* Cyan accent */
    --secondary: #a855f7;      /* Purple accent */
    --accent: #ec4899;         /* Pink accent */
    --dark-bg: #0a0e27;       /* Dark background */
    --dark-card: #1a1f3a;     /* Card background */
    --text-primary: #ffffff;   /* Main text */
    --text-secondary: #b0b8c1; /* Secondary text */
}
```
